Edges in a Tree

About 488 wordsAbout 6 min

2025-08-07

Question

Prove, by induction on nnn, that for all n≥1n \ge 1n≥1, every tree on nnn vertices has n−1n-1n−1 edges.

Use the fact that every tree with more than one vertex has a leaf (a vertex connected to only one edge).

Inductive Proof

We will prove the proposition P(n)P(n)P(n): "Every tree on nnn vertices has n−1n-1n−1 edges" for all integers n≥1n \ge 1n≥1.

  1. Base Case: P(1)P(1)P(1)

    We test the proposition for n=1n=1n=1. A tree with 111 vertex consists of a single node and has no edges. The formula states the number of edges should be n−1=1−1=0n-1 = 1-1 = 0n−1=1−1=0. Since a 1-vertex tree has 0 edges, the proposition P(1)P(1)P(1) is true.

  2. Inductive Hypothesis

    We assume that the proposition P(k)P(k)P(k) is true for some arbitrary integer k≥1k \ge 1k≥1. That is, we assume that every tree with kkk vertices has k−1k-1k−1 edges.

  3. Inductive Step: Proving P(k+1)P(k+1)P(k+1)

    Our goal is to prove that any tree with k+1k+1k+1 vertices must have (k+1)−1=k(k+1)-1 = k(k+1)−1=k edges.

    Let TTT be an arbitrary tree with k+1k+1k+1 vertices. Since k≥1k \ge 1k≥1, the number of vertices k+1≥2k+1 \ge 2k+1≥2. According to the provided hint, any tree with more than one vertex must have a leaf.

    Let's choose a leaf vertex from TTT and call it vvv. By definition, a leaf is connected to exactly one edge. Let's call this edge eee.

    Now, let's construct a new graph, T′T'T′, by removing the vertex vvv and the edge eee from the tree TTT.

    • T′T'T′ has (k+1)−1=k(k+1) - 1 = k(k+1)−1=k vertices.
    • T′T'T′ has one fewer edge than TTT.
    • Removing a leaf and its single connecting edge from a tree cannot create a cycle or disconnect the graph. Therefore, T′T'T′ is also a tree.

    Since T′T'T′ is a tree with kkk vertices, we can apply our Inductive Hypothesis to it. The hypothesis states that T′T'T′ must have k−1k-1k−1 edges.

    The original tree TTT was formed by adding the vertex vvv and the edge eee to T′T'T′. Thus, the number of edges in TTT is the number of edges in T′T'T′ plus one.

    Number of edges in T=(Edges in T′)+1=(k−1)+1=kT = (\text{Edges in } T') + 1 = (k-1) + 1 = kT=(Edges in T′)+1=(k−1)+1=k.

    This is exactly what we needed to show: a tree with k+1k+1k+1 vertices has kkk edges. Therefore, the proposition P(k+1)P(k+1)P(k+1) is true.

  4. Conclusion

    Since we have shown that the base case P(1)P(1)P(1) is true and that the truth of P(k)P(k)P(k) implies the truth of P(k+1)P(k+1)P(k+1), we conclude by the Principle of Mathematical Induction that every tree on nnn vertices has n−1n-1n−1 edges for all n≥1n \ge 1n≥1.
