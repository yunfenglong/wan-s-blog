Languages and terminology

About 609 wordsAbout 8 min

2025-07-29

Languages

  • Alphabet (Σ\SigmaΣ): A finite, non-empty set of symbols.
    • Example: Σ={a,b}\Sigma = \{a, b\}Σ={a,b}
  • Word: A finite string of symbols from an alphabet.
  • Empty Word (ϵ\epsilonϵ): The unique word with zero symbols.
  • Empty Set (∅\emptyset∅): The set with no elements.
  • Σ∗\Sigma^*Σ∗: The set of all possible words that can be formed from the alphabet Σ\SigmaΣ, including the empty word ϵ\epsilonϵ.

Key Language Examples

  • EVEN-EVEN: The language where each string has an even number of 'a's and an even number of 'b's.
    • Note: Since 0 is an even number, the empty word ϵ\epsilonϵ is in EVEN-EVEN.
  • DOUBLEWORD: The language of words formed by concatenating some word www with itself.
    • Formally: {ww∣w∈Σ∗}\{ww \mid w \in \Sigma^*\}{ww∣w∈Σ∗}.
    • Note: ϵ=ϵϵ\epsilon = \epsilon\epsilonϵ=ϵϵ, so ϵ\epsilonϵ is a doubleword.
  • PALINDROMES: The language of words that read the same forwards and backwards.
    • Formally: {w∣w=wR}\{w \mid w = w^R\}{w∣w=wR}, where wRw^RwR is the reverse of www.
  • ODD-ODD: The language where each string has an odd number of 'a's and an odd number of 'b's.
    • Example: The word "a" is not in ODD-ODD. It has 1 'a' (odd) but 0 'b's (even). The word "abb" is also not in ODD-ODD (1 'a', 2 'b's). The word "ab" is in ODD-ODD.

Definitions, theorems, and proofs

Theorem: DOUBLEWORD ⊆ EVEN-EVEN (Every doubleword is also an even-even word).

Proof: Let www be any word in Σ∗\Sigma^*Σ∗. Let the number of 'a's in www be nan_ana​ and the number of 'b's be nbn_bnb​. The corresponding word in DOUBLEWORD is wwwwww. The number of 'a's in wwwwww is 2na2n_a2na​ and the number of 'b's is 2nb2n_b2nb​. By definition, 2na2n_a2na​ and 2nb2n_b2nb​ are always even numbers. Therefore, any word in DOUBLEWORD is also in EVEN-EVEN.

Normal Forms

  • Literal: A propositional variable or its negation (e.g., AAA or ¬A\neg A¬A).
  • Disjunctive Normal Form (DNF): A disjunction (OR) of one or more clauses, where each clause is a conjunction (AND) of literals.
    • Structure: (A∧¬B)∨(C∧D)∨…(A \land \neg B) \lor (C \land D) \lor \dots(A∧¬B)∨(C∧D)∨…
  • Conjunctive Normal Form (CNF): A conjunction (AND) of one or more clauses, where each clause is a disjunction (OR) of literals.
    • Structure: (A∨¬B)∧(C∨D)∧…(A \lor \neg B) \land (C \lor D) \land \dots(A∨¬B)∧(C∨D)∧…

Statements with variables

The variables in the statement are free, which means they have not been assign any values.

Predicates definitions and terminology

  • Predicate: A statement with variables that becomes a proposition once the variables are given specific values from a domain. A predicate is called l-ary if it has k arguments.
    • Example: P(x)="x>10"P(x) = "x > 10"P(x)="x>10". P(12)P(12)P(12) is True, P(5)P(5)P(5) is False.
  • Universal Quantifier (∀\forall∀): Means "for all". ∀x,P(x)\forall x, P(x)∀x,P(x) asserts that P(x)P(x)P(x) is true for every xxx in the domain.
  • Existential Quantifier (∃\exists∃): Means "there exists". ∃x,P(x)\exists x, P(x)∃x,P(x) asserts that there is at least one xxx in the domain for which P(x)P(x)P(x) is true.

Propositions and logical operations

  • Proposition: A declarative sentence that is definitively True or False, but not both.
    • Paradox: "This statement is false" is a paradox, not a proposition, because its truth value cannot be determined without contradiction.
  • Tautology: A statement that is always true, no matter the truth values of its variables (e.g., p∨¬pp \lor \neg pp∨¬p).
  • Logical Equivalence (≡\equiv≡): Two statements are logically equivalent if they always have the same truth value.
    • Example: De Morgan's Law states ¬(p∧q)≡(¬p∨¬q)\neg(p \land q) \equiv (\neg p \lor \neg q)¬(p∧q)≡(¬p∨¬q).
