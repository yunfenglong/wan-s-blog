Pumping Lemma for Regular Languages

About 1492 wordsAbout 19 min

2025-08-07

Statement of the Pumping Lemma

Theorem: If LLL is a regular language, then there exists a constant p≥1p \geq 1p≥1 (the pumping length) such that any string s∈Ls \in Ls∈L with ∣s∣≥p|s| \geq p∣s∣≥p can be divided into three parts s=xyzs = xyzs=xyz satisfying:

  1. Length constraint: ∣xy∣≤p|xy| \leq p∣xy∣≤p
  2. Non-empty pump: ∣y∣>0|y| > 0∣y∣>0 (i.e., y≠ϵy \neq \epsilony=ϵ)
  3. Pumping condition: For all i≥0i \geq 0i≥0, xyiz∈Lxy^iz \in Lxyiz∈L

Important

The Pumping Lemma provides a necessary but not sufficient condition for regularity.

Understanding the Components

String Decomposition

For any sufficiently long string s∈Ls \in Ls∈L:

  • xxx: Prefix (possibly empty)
  • yyy: Pumpable substring (must be non-empty)
  • zzz: Suffix (possibly empty)

Pumping Operation

The lemma guarantees that we can "pump" the yyy part:

  • i=0i = 0i=0: Remove yyy (string becomes xzxzxz)
  • i=1i = 1i=1: Original string (string becomes xyzxyzxyz)
  • i=2i = 2i=2: Insert yyy once (string becomes xyyzxyyzxyyz)
  • i=ki = ki=k: Insert yyy k−1k-1k−1 times (string becomes xykzxy^kzxykz)

How to Use the Pumping Lemma

Strategy for Proving Non-regularity

  1. Assume LLL is regular and let ppp be its pumping length
  2. Choose a string s∈Ls \in Ls∈L with ∣s∣≥p|s| \geq p∣s∣≥p (carefully selected)
  3. Consider all possible decompositions s=xyzs = xyzs=xyz with ∣xy∣≤p|xy| \leq p∣xy∣≤p and ∣y∣>0|y| > 0∣y∣>0
  4. Show that for each decomposition, there exists some i≥0i \geq 0i≥0 such that xyiz∉Lxy^iz \notin Lxyiz∈/L
  5. Conclude that LLL is not regular

Proof Structure

To prove LLL is not regular:

Assume L is regular for contradiction
Let p be the pumping length
Choose s = (careful selection)
For all possible xyz decompositions:
    Find i such that xy^iz ∉ L
Therefore L is not regular

Common Examples

Example 1: L={anbn∣n≥0}L = \{a^n b^n \mid n \geq 0\}L={anbn∣n≥0}

Proof that LLL is not regular:

  1. Assume LLL is regular with pumping length ppp
  2. Choose s=apbp∈Ls = a^p b^p \in Ls=apbp∈L (note ∣s∣=2p≥p|s| = 2p \geq p∣s∣=2p≥p)
  3. Consider any decomposition s=xyzs = xyzs=xyz where ∣xy∣≤p|xy| \leq p∣xy∣≤p and ∣y∣>0|y| > 0∣y∣>0
    • Since ∣xy∣≤p|xy| \leq p∣xy∣≤p, xyxyxy consists only of aaa's
    • Let y=aky = a^ky=ak where k≥1k \geq 1k≥1
  4. Pump with i=2i = 2i=2: xy2z=ap+kbpxy^2z = a^{p+k} b^pxy2z=ap+kbp
    • This has p+kp+kp+k aaa's but only ppp bbb's
    • Since k≥1k \geq 1k≥1, p+k≠pp+k \neq pp+k=p
    • Therefore ap+kbp∉La^{p+k} b^p \notin Lap+kbp∈/L
  5. Conclusion: LLL is not regular

Example 2: L={ww∣w∈{a,b}∗}L = \{ww \mid w \in \{a, b\}^*\}L={ww∣w∈{a,b}∗}

Proof that LLL is not regular:

  1. Assume LLL is regular with pumping length ppp
  2. Choose s=apbapb∈Ls = a^p b a^p b \in Ls=apbapb∈L (where w=apbw = a^p bw=apb)
  3. Consider any decomposition s=xyzs = xyzs=xyz with ∣xy∣≤p|xy| \leq p∣xy∣≤p and ∣y∣>0|y| > 0∣y∣>0
    • xyxyxy must be entirely within the first apa^pap sequence
    • Let y=aky = a^ky=ak where k≥1k \geq 1k≥1
  4. Pump with i=0i = 0i=0: xz=ap−kbapbxz = a^{p-k} b a^p bxz=ap−kbapb
    • This cannot be written as wwwwww for any www
    • The first half would need to be a(p−k)/2ba^{(p-k)/2} ba(p−k)/2b but (p−k)/2(p-k)/2(p−k)/2 may not be integer
  5. Conclusion: LLL is not regular

Example 3: L={an2∣n≥0}L = \{a^{n^2} \mid n \geq 0\}L={an2∣n≥0}

Proof that LLL is not regular:

  1. Assume LLL is regular with pumping length ppp
  2. Choose s=ap2∈Ls = a^{p^2} \in Ls=ap2∈L
  3. Consider decomposition s=xyzs = xyzs=xyz with ∣xy∣≤p|xy| \leq p∣xy∣≤p and ∣y∣=k|y| = k∣y∣=k where k≥1k \geq 1k≥1
  4. Pump with i=2i = 2i=2: xy2z=ap2+kxy^2z = a^{p^2 + k}xy2z=ap2+k
    • We need p2+k=m2p^2 + k = m^2p2+k=m2 for some integer mmm
    • But p2<p2+k≤p2+p<(p+1)2=p2+2p+1p^2 < p^2 + k \leq p^2 + p < (p+1)^2 = p^2 + 2p + 1p2<p2+k≤p2+p<(p+1)2=p2+2p+1
    • Since k≤pk \leq pk≤p, p2+kp^2 + kp2+k cannot be a perfect square
  5. Conclusion: LLL is not regular

Example 4: L={aibjck∣i=j or j=k}L = \{a^i b^j c^k \mid i = j \text{ or } j = k\}L={aibjck∣i=j or j=k}

Proof that LLL is not regular:

  1. Assume LLL is regular with pumping length ppp
  2. Choose s=apbpcp+1∈Ls = a^p b^p c^{p+1} \in Ls=apbpcp+1∈L (satisfies i=ji = ji=j)
  3. Consider any decomposition s=xyzs = xyzs=xyz with ∣xy∣≤p|xy| \leq p∣xy∣≤p and ∣y∣>0|y| > 0∣y∣>0
    • Since ∣xy∣≤p|xy| \leq p∣xy∣≤p, yyy consists only of aaa's
    • Let y=aky = a^ky=ak where k≥1k \geq 1k≥1
  4. Pump with i=0i = 0i=0: xz=ap−kbpcp+1xz = a^{p-k} b^p c^{p+1}xz=ap−kbpcp+1
    • For this to be in LLL, we need either:
      • p−k=pp-k = pp−k=p (impossible since k≥1k \geq 1k≥1), or
      • p=p+1p = p+1p=p+1 (impossible)
    • Therefore xz∉Lxz \notin Lxz∈/L
  5. Conclusion: LLL is not regular

This example shows how the pumping lemma can handle languages with multiple conditions.

Common Mistakes and Pitfalls

Incorrect String Selection

Mistake: Choosing strings that are too short or don't create the right imbalance

Solution: Choose strings where the pumped part creates a clear violation

Wrong Pumping Index

Mistake: Always using i=2i = 2i=2 when i=0i = 0i=0 might be more effective

Solution: Consider different values of iii based on the language structure

Ignoring All Decompositions

Mistake: Only considering one specific decomposition

Solution: Must consider all possible decompositions satisfying the constraints

Variations and Extensions

Pumping Lemma for Context-Free Languages

The pumping lemma for context-free languages is more complex but follows a similar approach:

  • Strings can be divided into five parts: uvwxyuvwxyuvwxy
  • Two parts can be pumped independently

Ogden's Lemma

A stronger version that allows marking positions in the string to pump.

Practical Applications

Compiler Design

  • Proving that certain language constructs cannot be handled by regular grammars
  • Understanding limitations of lexical analysis

Formal Language Theory

  • Classifying languages within the Chomsky hierarchy
  • Understanding the boundaries of different language classes

Limitations

What the Pumping Lemma Cannot Do

  1. Prove regularity: The lemma only provides a necessary condition, not sufficient
  2. Give the pumping length: The lemma doesn't tell us what ppp is
  3. Handle all cases: Some non-regular languages may still satisfy the pumping lemma for certain pumping lengths

When the Pumping Lemma Fails

Some non-regular languages satisfy the pumping lemma:

  • L={anbmcndm∣n,m≥0}L = \{a^n b^m c^n d^m \mid n, m \geq 0\}L={anbmcndm∣n,m≥0} (not regular but satisfies pumping lemma for certain cases)

Summary

The Pumping Lemma is a powerful tool for proving that languages are not regular. By:

  • Understanding the decomposition and pumping conditions
  • Carefully selecting test strings
  • Systematically analyzing all possible decompositions
  • Choosing appropriate pumping indices

We can demonstrate that many interesting languages (like balanced parentheses, equal numbers of symbols, etc.) cannot be recognized by finite automata, highlighting the limitations of regular languages and motivating the study of more powerful language classes.

Remember that the Pumping Lemma is a proof by contradiction technique - we assume regularity and show that it leads to a contradiction by finding strings that cannot be pumped properly.

Problem-Solving Skills: Picking s and i

  1. Analyze constraints of L

    • Identify the invariant that pumping will likely break (e.g., equal counts, mirrored halves, forbidden substrings at boundaries).

  2. Choose s with |s| ≥ p

    • Place the first p symbols to force y within a controlled region (often a block of the same symbol).

  3. Quantify all decompositions

    • Because |xy| ≤ p, y lies in the prefix you control; write y’s general form.

  4. Pick i smartly

    • Use i=0 to delete structure or i=2 to increase a count; choose whichever breaks the invariant.

  5. Conclude by contradiction

    • Show at least one i yields a string not in L for every legal decomposition.

Additional Worked Examples

L = { strings over {a,b} with exactly one b }

Assume regular, pumping length p. Choose s = a^p b a^p. For any |xy| ≤ p, y = a^k with k ≥ 1. Pump i=0 ⇒ a^{p-k} b a^p has still one b, but length of a’s changes. That alone doesn’t break the spec. Instead pump i=2 ⇒ a^{p+k} b a^p still exactly one b. This language is actually regular; pumping does not lead to a contradiction. Use this as a reminder to test closure intuition before applying pumping.

L = { w ∈ {0,1}^* | #0(w) = #1(w) }

Choose s = 0^p 1^p. y lies in 0^p, so y = 0^k, k ≥ 1. Pump i=2 ⇒ 0{p+k}1p, counts differ ⇒ not in L.

Strategy Comparisons

Pumping Lemma:active

Use to prove non-regularity by contradiction.

Myhill–Nerode

Show infinitely many distinguishable prefixes/suffixes to prove no finite index, hence non-regular.

Closure Tricks

Transform with closures (complement, intersection, homomorphisms) to/from known non-regular languages.

Tips

If pumping seems messy, try Myhill–Nerode: fix a family of distinguishing continuations and show they separate infinitely many prefixes.
