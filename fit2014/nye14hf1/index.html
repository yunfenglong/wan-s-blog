Hereditary Languages

About 411 wordsAbout 5 min

2025-08-07

Question

A language LLL is called hereditary if it has the following property:

For every nonempty string zzz in LLL, there is a character in zzz which can be deleted from zzz to give another string in LLL.

Prove by contradiction that every nonempty hereditary language contains the empty string.

Answer

Here is a proof by contradiction.

1. Proposition to Prove: Let LLL be a language. If LLL is nonempty and hereditary, then the empty string, ϵ\epsilonϵ, is in LLL.

2. Assumption for Contradiction: Assume the proposition is false. This means there exists a language LLL that is nonempty and hereditary, but does not contain the empty string (ϵ∉L\epsilon \notin Lϵ∈/L).

3. Argument:

  • Since we assume LLL is nonempty, we can choose a string from it. Let's find a string in LLL with the shortest possible length. Since the set of lengths of strings in LLL is a nonempty set of non-negative integers, by the Well-Ordering Principle, a string with a minimum length must exist. Let's call this string sss.
  • By our assumption, we know that ϵ∉L\epsilon \notin Lϵ∈/L. This means that sss cannot be the empty string, because if it were, ϵ\epsilonϵ would be in LLL. Therefore, sss must be a nonempty string.
  • Now, we use the fact that LLL is a hereditary language. The property of being hereditary states that for any nonempty string in LLL (which sss is), we can delete a character to create a new string that is also in LLL.
  • Let's create such a new string by deleting one character from sss. Let's call this new string s′s's′. By the hereditary property, s′∈Ls' \in Ls′∈L.
  • The length of s′s's′ is exactly one less than the length of sss. So, length(s′s's′) < length(sss).
  • This leads to a contradiction. We chose sss to be a string in LLL with the minimum possible length. However, we have just constructed another string, s′s's′, which is also in LLL but is shorter than sss. This contradicts the definition of sss as the shortest string in LLL.

4. Conclusion: Our initial assumption—that a nonempty, hereditary language LLL can exist without containing the empty string—must be false.

Therefore, we conclude that every nonempty hereditary language must contain the empty string.
