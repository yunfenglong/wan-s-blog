Mimsy Boolean Expressions in CNF

About 1082 wordsAbout 14 min

2025-08-07

Question

A boolean expression in Conjunctive Normal Form (CNF) is mimsy if every clause contains a variable that appears in no earlier clause. So, if φ=C1∧C2∧⋯∧Ci∧⋯∧Cm\varphi = C_1 \wedge C_2 \wedge \dots \wedge C_i \wedge \dots \wedge C_mφ=C1​∧C2​∧⋯∧Ci​∧⋯∧Cm​ where each CiC_iCi​ is a disjunction of literals, then φ\varphiφ is mimsy if, for each i, there is a variable xix_ixi​ that appears in CiC_iCi​ (either as xix_ixi​ or negated ¬xi\neg x_i¬xi​) but does not appear in any of C1,…,Ci−1C_1, \dots, C_{i-1}C1​,…,Ci−1​.

For example, the following expression α\alphaα is mimsy: α:=(x∨¬y)∧(y∨¬z)\alpha := (x \lor \neg y) \wedge (y \lor \neg z)α:=(x∨¬y)∧(y∨¬z). This is because the first clause, C1=x∨¬yC_1 = x \lor \neg yC1​=x∨¬y, includes at least one variable that does not appear in any earlier clauses, and the second clause, C2=y∨¬zC_2 = y \lor \neg zC2​=y∨¬z, includes a variable, z, that does not appear in C1C_1C1​.

On the other hand, the following expression β\betaβ is not mimsy: β:=(x∨y)∧(x∨y)\beta := (x \lor y) \wedge (x \lor y)β:=(x∨y)∧(x∨y). This is because all the variables in the second clause also appeared in the first clause.

A Boolean expression is satisfiable if there is an assignment of truth values to its variables that makes the expression True.

Prove the following statement, by mathematical induction:

For all m≥1m \ge 1m≥1, every mimsy Boolean expression in CNF with mmm clauses is satisfiable.

(a) Pre-proof exploration: (i) Show that each of the example expressions above, α\alphaα and β\betaβ, is satisfiable. For each of them, you just need to give one truth assignment to its variables that satisfies it. (ii) Construct an expression in CNF, with at least two literals in each clause, that is not mimsy and not satisfiable.

(b) Inductive basis: prove the statement (*) for m=1m=1m=1.

(c) Given a mimsy Boolean expression φ\varphiφ with m+1m+1m+1 clauses, show how to construct from it a mimsy Boolean expression φ−\varphi^-φ− with just m clauses that will help with your proof.

(d) What does your Inductive Hypothesis tell you about φ−\varphi^-φ−?

(e) Using the fact that φ\varphiφ is mimsy, and your answer to (d), show that φ\varphiφ has a satisfying truth assignment.

(f) When drawing your final conclusion, don't forget to briefly state that you are using the Principle of Mathematical Induction!

Answer

(a) Pre-proof exploration:

(i) Satisfiability of α\alphaα and β\betaβ

  • For the expression α:=(x∨¬y)∧(y∨¬z)\alpha := (x \lor \neg y) \wedge (y \lor \neg z)α:=(x∨¬y)∧(y∨¬z), a satisfying truth assignment is x = True, y = True, z = True.

    • (True∨¬True)∧(True∨¬True)(True \lor \neg True) \wedge (True \lor \neg True)(True∨¬True)∧(True∨¬True)
    • (True∨False)∧(True∨False)(True \lor False) \wedge (True \lor False)(True∨False)∧(True∨False)
    • True∧TrueTrue \wedge TrueTrue∧True, which is True.

  • For the expression β:=(x∨y)∧(x∨y)\beta := (x \lor y) \wedge (x \lor y)β:=(x∨y)∧(x∨y), a satisfying truth assignment is x = True, y = False.

    • (True∨False)∧(True∨False)(True \lor False) \wedge (True \lor False)(True∨False)∧(True∨False)
    • True∧TrueTrue \wedge TrueTrue∧True, which is True.

(ii) Non-mimsy and non-satisfiable expression

An example of an expression in CNF with at least two literals in each clause that is not mimsy and not satisfiable is: (x∨y)∧(¬x∨¬y)∧(x∨¬y)∧(¬x∨y)(x \lor y) \wedge (\neg x \lor \neg y) \wedge (x \lor \neg y) \wedge (\neg x \lor y)(x∨y)∧(¬x∨¬y)∧(x∨¬y)∧(¬x∨y)

  • Not Mimsy: All variables in the subsequent clauses (xxx and yyy) have already appeared in the first clause.
  • Not Satisfiable: There is no truth assignment for xxx and yyy that will make all four clauses true simultaneously. For any of the four possible assignments (T,T), (T,F), (F,T), (F,F), one of the clauses will evaluate to False.

(b) Inductive basis:

We must prove the statement for m=1m=1m=1: "every mimsy Boolean expression in CNF with 1 clause is satisfiable". An expression with one clause is just C1C_1C1​. By definition, any expression with one clause is mimsy because there are no earlier clauses for its variables to have appeared in. A clause is a disjunction of literals. We can satisfy the clause by choosing a truth assignment that makes at least one of its literals true. For any variable xix_ixi​ in C1C_1C1​, if the literal is xix_ixi​, we set xix_ixi​ to True; if the literal is ¬xi\neg x_i¬xi​, we set xix_ixi​ to False. This satisfies C1C_1C1​. Thus, the statement holds for m=1m=1m=1.

(c) Construction of φ−\varphi^-φ−

Given a mimsy Boolean expression φ\varphiφ with m+1m+1m+1 clauses, φ=C1∧⋯∧Cm∧Cm+1\varphi = C_1 \wedge \dots \wedge C_m \wedge C_{m+1}φ=C1​∧⋯∧Cm​∧Cm+1​. We construct φ−\varphi^-φ− by removing the last clause: φ−=C1∧⋯∧Cm\varphi^- = C_1 \wedge \dots \wedge C_mφ−=C1​∧⋯∧Cm​. This new expression φ−\varphi^-φ− has mmm clauses and is also mimsy, as the condition for being mimsy holds for each of its clauses C1,…,CmC_1, \dots, C_mC1​,…,Cm​ just as it did in the original expression φ\varphiφ.

(d) Implication of the Inductive Hypothesis

The Inductive Hypothesis (IH) assumes that every mimsy Boolean expression in CNF with mmm clauses is satisfiable. Since φ−\varphi^-φ− is a mimsy Boolean expression with mmm clauses, the IH tells us that φ−\varphi^-φ− is satisfiable. This means a truth assignment exists that makes φ−\varphi^-φ− true.

(e) Proving satisfiability for φ\varphiφ

We know φ=φ−∧Cm+1\varphi = \varphi^- \wedge C_{m+1}φ=φ−∧Cm+1​ and that there is a truth assignment, AAA, that satisfies φ−\varphi^-φ−. Because the original expression φ\varphiφ is mimsy, the clause Cm+1C_{m+1}Cm+1​ must contain a variable, xm+1x_{m+1}xm+1​, that does not appear in any of the preceding clauses C1,…,CmC_1, \dots, C_mC1​,…,Cm​.

This means the variable xm+1x_{m+1}xm+1​ is not in φ−\varphi^-φ−, and the assignment AAA does not assign it a value. We can extend assignment AAA by assigning a truth value to xm+1x_{m+1}xm+1​ without affecting the truth of φ−\varphi^-φ−. We choose a value for xm+1x_{m+1}xm+1​ that satisfies Cm+1C_{m+1}Cm+1​:

  • If Cm+1C_{m+1}Cm+1​ contains the literal xm+1x_{m+1}xm+1​, we set xm+1=x_{m+1} =xm+1​= True.
  • If Cm+1C_{m+1}Cm+1​ contains the literal ¬xm+1\neg x_{m+1}¬xm+1​, we set xm+1=x_{m+1} =xm+1​= False.

This extension makes Cm+1C_{m+1}Cm+1​ true. Since the assignment still satisfies φ−\varphi^-φ−, the full extended assignment satisfies φ=φ−∧Cm+1\varphi = \varphi^- \wedge C_{m+1}φ=φ−∧Cm+1​. Therefore, φ\varphiφ is satisfiable.

(f) Conclusion

We have shown that the base case (m=1m=1m=1) is true, and we have shown that if the statement holds for any integer m≥1m \ge 1m≥1, it also holds for m+1m+1m+1. Therefore, by the Principle of Mathematical Induction, we conclude that every mimsy Boolean expression in CNF with mmm clauses is satisfiable for all m≥1m \ge 1m≥1.
