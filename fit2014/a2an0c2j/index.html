Question 1: Truth Tables, DNF, and CNF

Scenario: A server has three critical services: Apache, BIND, and Cron. For the system to be considered "stable," at least two of these services must be running.

Let's represent the state of the services with propositions A, B, and C, where True means the service is running. We can construct a truth table to define the "stable" function.

ABCSystem Stable?DNF Clause (if True)CNF Clause (from False rows)
FFFF(A∨B∨C)(A \lor B \lor C)(A∨B∨C)
FFTF(A∨B∨¬C)(A \lor B \lor \neg C)(A∨B∨¬C)
FTFF(A∨¬B∨C)(A \lor \neg B \lor C)(A∨¬B∨C)
FTTT(¬A∧B∧C)(\neg A \land B \land C)(¬A∧B∧C)
TFFF(¬A∨B∨C)(\neg A \lor B \lor C)(¬A∨B∨C)
TFTT(A∧¬B∧C)(A \land \neg B \land C)(A∧¬B∧C)
TTFT(A∧B∧¬C)(A \land B \land \neg C)(A∧B∧¬C)
TTTT(A∧B∧C)(A \land B \land C)(A∧B∧C)

Deriving the DNF (Disjunctive Normal Form)

The DNF is the disjunction (OR) of the minterms for each row where the function's output is True.

  • DNF Result: (¬A∧B∧C)∨(A∧¬B∧C)∨(A∧B∧¬C)∨(A∧B∧C)(\neg A \land B \land C) \lor (A \land \neg B \land C) \lor (A \land B \land \neg C) \lor (A \land B \land C)(¬A∧B∧C)∨(A∧¬B∧C)∨(A∧B∧¬C)∨(A∧B∧C)

This DNF expression is true if exactly two services are running or if all three are running.

Deriving the CNF (Conjunctive Normal Form)

The CNF is the conjunction (AND) of maxterms derived from the rows where the function's output is False. For each 'False' row, we create a disjunction (OR) of the negated literals and apply De Morgan's laws.

  1. Row 1 (FFF): The condition is (¬A∧¬B∧¬C)(\neg A \land \neg B \land \neg C)(¬A∧¬B∧¬C). Negating this gives (A∨B∨C)(A \lor B \lor C)(A∨B∨C).
  2. Row 2 (FFT): The condition is (¬A∧¬B∧C)(\neg A \land \neg B \land C)(¬A∧¬B∧C). Negating this gives (A∨B∨¬C)(A \lor B \lor \neg C)(A∨B∨¬C).
  3. Row 3 (FTF): The condition is (¬A∧B∧¬C)(\neg A \land B \land \neg C)(¬A∧B∧¬C). Negating this gives (A∨¬B∨C)(A \lor \neg B \lor C)(A∨¬B∨C).
  4. Row 5 (TFF): The condition is (A∧¬B∧¬C)(A \land \neg B \land \neg C)(A∧¬B∧¬C). Negating this gives (¬A∨B∨C)(\neg A \lor B \lor C)(¬A∨B∨C).
  • CNF Result: (A∨B∨C)∧(A∨B∨¬C)∧(A∨¬B∨C)∧(¬A∨B∨C)(A \lor B \lor C) \land (A \lor B \lor \neg C) \land (A \lor \neg B \lor C) \land (\neg A \lor B \lor C)(A∨B∨C)∧(A∨B∨¬C)∧(A∨¬B∨C)∧(¬A∨B∨C)

This CNF expression logically states that we cannot have zero services running or only one service running.

Question 2: Properties of Formal Languages

Claim: All words in the language EVEN-EVEN have an even length.

Defining the EVEN-EVEN Language

The EVEN-EVEN language consists of strings over the alphabet {a,b}\{a, b\}{a,b} where the number of 'a's is even (0, 2, 4, ...) and the number of 'b's is also even (0, 2, 4, ...).

  • Let ∣w∣a|w|_a∣w∣a​ be the count of 'a's in a string www.
  • Let ∣w∣b|w|_b∣w∣b​ be the count of 'b's in a string www.

For a string www to be in EVEN-EVEN, ∣w∣a|w|_a∣w∣a​ must be even and ∣w∣b|w|_b∣w∣b​ must be even.

Examples: aa, bb, aabb, aaaabb, bbbb, and the empty string ϵ\epsilonϵ.

Mathematical Proof

The total length of any string www is ∣w∣=∣w∣a+∣w∣b|w| = |w|_a + |w|_b∣w∣=∣w∣a​+∣w∣b​.

  1. Since ∣w∣a|w|_a∣w∣a​ is an even number, it can be written as 2k2k2k for some non-negative integer kkk.
  2. Since ∣w∣b|w|_b∣w∣b​ is an even number, it can be written as 2m2m2m for some non-negative integer mmm.

Now, substitute these into the formula for the total length:

∣w∣=(2k)+(2m)|w| = (2k) + (2m)∣w∣=(2k)+(2m)∣w∣=2(k+m)|w| = 2(k + m)∣w∣=2(k+m)

Since kkk and mmm are integers, their sum (k+m)(k + m)(k+m) is also an integer. The expression 2(k+m)2(k + m)2(k+m) is, by definition, an even number.

Therefore, every word in the EVEN-EVEN language must have an even length.

Question 3: Proof of a Language Superset

Let EQUAL be the language of strings over {a,b}\{a,b\}{a,b} that contain an equal number of a’s and b’s. Let ~ODD-ODD be the complement of the ODD-ODD language (i.e., strings that do not have both an odd number of a's and an odd number of b's).

Prove that EQUAL ⊆ ~ODD-ODD.

Proof

To prove this, we must show that any arbitrary string w∈EQUALw \in \text{EQUAL}w∈EQUAL must also be in O˜DD-ODD\text{\~ODD-ODD}O˜DD-ODD.

  1. Let www be an arbitrary string in the language EQUAL.

  2. By the definition of EQUAL, the number of 'a's in www is equal to the number of 'b's in www. Let's call this number kkk.

    • ∣w∣a=k|w|_a = k∣w∣a​=k
    • ∣w∣b=k|w|_b = k∣w∣b​=k

  3. The definition of ~ODD-ODD means a string is in this language if ∣w∣a|w|_a∣w∣a​ is even OR ∣w∣b|w|_b∣w∣b​ is even.

  4. We analyze two cases for the value of kkk:

    • Case 1: kkk is an even number. If kkk is even, then ∣w∣a=k|w|_a = k∣w∣a​=k is even and ∣w∣b=k|w|_b = k∣w∣b​=k is also even. Since ∣w∣a|w|_a∣w∣a​ is even, the condition for www to be in ~ODD-ODD is satisfied.
    • Case 2: kkk is an odd number. If kkk is odd, then ∣w∣a=k|w|_a = k∣w∣a​=k is odd and ∣w∣b=k|w|_b = k∣w∣b​=k is also odd. This means www has an odd number of 'a's and an odd number of 'b's. This is the exact definition of a string in the ODD-ODD language.

  5. Let's re-examine our premise. The proof seems to fail in Case 2. Let's reconsider the properties. The length of any string in EQUAL is ∣w∣=∣w∣a+∣w∣b=k+k=2k|w| = |w|_a + |w|_b = k + k = 2k∣w∣=∣w∣a​+∣w∣b​=k+k=2k. This means every string in EQUAL has an even length.

    Let's restart the proof with this insight.

    • Let w∈EQUALw \in \text{EQUAL}w∈EQUAL. Then ∣w∣a=∣w∣b=k|w|_a = |w|_b = k∣w∣a​=∣w∣b​=k.
    • To be in ODD-ODD, ∣w∣a|w|_a∣w∣a​ must be odd and ∣w∣b|w|_b∣w∣b​ must be odd.
    • If ∣w∣a|w|_a∣w∣a​ and ∣w∣b|w|_b∣w∣b​ were both odd, then kkk would be an odd number.
    • However, if kkk is odd, ∣w∣a|w|_a∣w∣a​ is odd and ∣w∣b|w|_b∣w∣b​ is odd. This fits the definition of ODD-ODD. Example: ab. ∣w∣a=1,∣w∣b=1|w|_a=1, |w|_b=1∣w∣a​=1,∣w∣b​=1. This is in EQUAL and also in ODD-ODD.
    • This means ab is NOT in ~ODD-ODD.

Therefore, the statement EQUAL ⊆ ~ODD-ODD is FALSE. The string ab is a counterexample, as it is in EQUAL but not in ~ODD-ODD.

(This demonstrates the importance of verifying claims and finding counterexamples).

Let's prove a different, correct statement: If a palindrome has an even length, it is in ~ODD-ODD.

  • Let www be a palindrome with even length ∣w∣=2k|w| = 2k∣w∣=2k.
  • We can write w=uuRw = uu^Rw=uuR for some string uuu of length kkk.
  • The number of 'a's in www is ∣w∣a=∣u∣a+∣uR∣a=2∣u∣a|w|_a = |u|_a + |u^R|_a = 2|u|_a∣w∣a​=∣u∣a​+∣uR∣a​=2∣u∣a​. This is an even number.
  • Since ∣w∣a|w|_a∣w∣a​ is even, www cannot have an odd number of 'a's AND an odd number of 'b's.
  • Therefore, w∈O˜DD-ODDw \in \text{\~ODD-ODD}w∈O˜DD-ODD.

Question 4: Proof by Truth Table

Prove the logical equivalence of De Morgan's Law: ¬(P∨Q)≡(¬P∧¬Q)\neg(P \lor Q) \equiv (\neg P \land \neg Q)¬(P∨Q)≡(¬P∧¬Q)

PQP∨QP \lor QP∨Q¬(P∨Q)\neg(P \lor Q)¬(P∨Q)¬P\neg P¬P¬Q\neg Q¬Q¬P∧¬Q\neg P \land \neg Q¬P∧¬Q
TTTFFFF
TFTFFTF
FTTFTFF
FFFTTTT

Since the columns for ¬(P∨Q)\neg(P \lor Q)¬(P∨Q) and (¬P∧¬Q)(\neg P \land \neg Q)(¬P∧¬Q) are identical for all possible truth values of P and Q, the two expressions are logically equivalent.

Question 5: Proof by Derivation

Using the distributive law A∨(B∧C)≡(A∨B)∧(A∨C)A \lor (B \land C) \equiv (A \lor B) \land (A \lor C)A∨(B∧C)≡(A∨B)∧(A∨C), prove the absorption law: P∨(P∧Q)≡PP \lor (P \land Q) \equiv PP∨(P∧Q)≡P.

You may also use the following axioms:

  • Identity Law: P∨False≡PP \lor \text{False} \equiv PP∨False≡P
  • Complementation Law: P∧¬P≡FalseP \land \neg P \equiv \text{False}P∧¬P≡False
  • Domination Law: P∨True≡TrueP \lor \text{True} \equiv \text{True}P∨True≡True
  • Tautology: A≡A∧TrueA \equiv A \land \text{True}A≡A∧True

  1. Start with the Left-Hand Side (LHS):P∨(P∧Q)P \lor (P \land Q)P∨(P∧Q)

  2. Apply the distributive law: A∨(B∧C)≡(A∨B)∧(A∨C)A \lor (B \land C) \equiv (A \lor B) \land (A \lor C)A∨(B∧C)≡(A∨B)∧(A∨C). Here, A=PA = PA=P, B=PB = PB=P, and C=QC = QC=Q. P∨(P∧Q)≡(P∨P)∧(P∨Q)P \lor (P \land Q) \equiv (P \lor P) \land (P \lor Q)P∨(P∧Q)≡(P∨P)∧(P∨Q)

  3. Apply the Idempotent Law (P∨P≡PP \lor P \equiv PP∨P≡P): ≡P∧(P∨Q)\equiv P \land (P \lor Q)≡P∧(P∨Q)

    (This proves the other absorption law, P∧(P∨Q)≡PP \land (P \lor Q) \equiv PP∧(P∨Q)≡P. Let's try another way to prove the original statement).

Let's try a different approach starting from the LHS.

  1. Start with the LHS:P∨(P∧Q)P \lor (P \land Q)P∨(P∧Q)

  2. Use the Tautology axiom to write PPP as P∧TrueP \land \text{True}P∧True: ≡(P∧True)∨(P∧Q)\equiv (P \land \text{True}) \lor (P \land Q)≡(P∧True)∨(P∧Q)

  3. Apply the distributive law (A∧B)∨(A∧C)≡A∧(B∨C)(A \land B) \lor (A \land C) \equiv A \land (B \lor C)(A∧B)∨(A∧C)≡A∧(B∨C) in reverse. Here, A=PA = PA=P, B=TrueB = \text{True}B=True, C=QC = QC=Q. ≡P∧(True∨Q)\equiv P \land (\text{True} \lor Q)≡P∧(True∨Q)

  4. Apply the Domination Law (True∨Q≡True\text{True} \lor Q \equiv \text{True}True∨Q≡True): ≡P∧True\equiv P \land \text{True}≡P∧True

  5. Apply the Identity Law (P∧True≡PP \land \text{True} \equiv PP∧True≡P): ≡P\equiv P≡P

We have shown that P∨(P∧Q)≡PP \lor (P \land Q) \equiv PP∨(P∧Q)≡P.

Question 6: Translating English to CNF

A team of three students, Alice, Bob, and Carol, are working on a project. Let the propositions A, B, and C mean that the respective student completed their part of the work.

Write a proposition in Conjunctive Normal Form (CNF) for each statement.

  1. At least one student completed their work.
    • (A∨B∨C)(A \lor B \lor C)(A∨B∨C)
  2. Carol and Alice did not both complete their work.
    • (¬C∨¬A)(\neg C \lor \neg A)(¬C∨¬A)
  3. Exactly one student completed their work.
    • This means "at least one" AND "at most one".
    • At least one: (A∨B∨C)(A \lor B \lor C)(A∨B∨C)
    • At most one (no two worked): (¬A∨¬B)∧(¬A∨¬C)∧(¬B∨¬C)(\neg A \lor \neg B) \land (\neg A \lor \neg C) \land (\neg B \lor \neg C)(¬A∨¬B)∧(¬A∨¬C)∧(¬B∨¬C)
    • Result: (A∨B∨C)∧(¬A∨¬B)∧(¬A∨¬C)∧(¬B∨¬C)(A \lor B \lor C) \land (\neg A \lor \neg B) \land (\neg A \lor \neg C) \land (\neg B \lor \neg C)(A∨B∨C)∧(¬A∨¬B)∧(¬A∨¬C)∧(¬B∨¬C)
  4. At most two students completed their work.
    • This is the same as saying "it's not the case that all three completed their work": ¬(A∧B∧C)\neg(A \land B \land C)¬(A∧B∧C)
    • Result: (¬A∨¬B∨¬C)(\neg A \lor \neg B \lor \neg C)(¬A∨¬B∨¬C)
  5. If Bob completed his work, then Alice also completed her work.
    • B  ⟹  AB \implies AB⟹A
    • Result: (¬B∨A)(\neg B \lor A)(¬B∨A)

Question 7: Predicate Logic Translation

Using the function capital(c) (the capital of country c), the predicate isNorthOf(p1, p2) (place p1 is north of place p2), and the constants malaysia and thailand, translate the following sentences into Predicate Logic. The universe of discourse is "all places and countries".

  1. The capital of Malaysia is north of the capital of Thailand.
    • isNorthOf(capital(malaysia), capital(thailand))
  2. There is a country whose capital is north of Kuala Lumpur.
    • Let kualaLumpur be a constant representing the city.
    • ∃x isNorthOf(capital(x),kualaLumpur)\exists x \space isNorthOf(capital(x), kualaLumpur)∃x isNorthOf(capital(x),kualaLumpur)
  3. Every place is north of some other place.
    • ∀x ∃y (isNorthOf(x,y)∧x≠y)\forall x \space \exists y \space (isNorthOf(x, y) \land x \neq y)∀x ∃y (isNorthOf(x,y)∧x=y)
  4. Every city that is north of Bangkok is also north of Singapore.
    • Let bangkok and singapore be constants.
    • ∀x isNorthOf(x,bangkok)  ⟹  isNorthOf(x,singapore)\forall x \space isNorthOf(x, bangkok) \implies isNorthOf(x, singapore)∀x isNorthOf(x,bangkok)⟹isNorthOf(x,singapore)

Question 8: Language Properties in Predicate Logic

Suppose you have access to the function ∣x∣|x|∣x∣ (length of string xxx), relations like = and ∈, and a language LLL over the alphabet {a,b}\{a, b\}{a,b}.

  1. Write a logical statement that is True if and only if LLL contains the empty string ϵ\epsilonϵ.

    • The statement is simply that the empty string is an element of the set L.
    • ϵ∈L\epsilon \in Lϵ∈L
    • Alternatively, using a quantifier: ∃x (x∈L∧∣x∣=0)\exists x \space (x \in L \land |x| = 0)∃x (x∈L∧∣x∣=0)

  2. Write a logical statement that is True if and only if LLL contains a finite number of strings of length 5.

    • This means there exists some number kkk that is an upper bound on the count of strings of length 5 in L. This is more complex than defining if the entire language is finite. A simpler interpretation is to state there isn't an infinite number of such strings. We can't easily express "the count of" in basic predicate logic.
    • A better approach for this level might be: Write a statement that is True if and only if L contains at least one string of every possible length.
    • ∀n∃x(x∈L∧∣x∣=n)\forall n \exists x (x \in L \land |x| = n)∀n∃x(x∈L∧∣x∣=n)
