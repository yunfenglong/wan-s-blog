(a) Prove, by induction on nnn, that for all n≥3n \ge 3n≥3, n!≤(n−1)nn! \le (n-1)^nn!≤(n−1)n

Proposition $P(n)$

The statement to prove is P(n):n!≤(n−1)nP(n): n! \le (n-1)^nP(n):n!≤(n−1)n for all integers n≥3n \ge 3n≥3.

Inductive Proof

  1. Base Case: P(3)P(3)P(3) For n=3n=3n=3, we check if 3!≤(3−1)33! \le (3-1)^33!≤(3−1)3.

    • 3!=63! = 63!=6
    • (3−1)3=23=8(3-1)^3 = 2^3 = 8(3−1)3=23=8 Since 6≤86 \le 86≤8, the proposition P(3)P(3)P(3) holds true.

  2. Inductive Hypothesis Assume that P(k)P(k)P(k) is true for some arbitrary integer k≥3k \ge 3k≥3. That is, we assume: k!≤(k−1)kk! \le (k-1)^kk!≤(k−1)k.

  3. Inductive Step: Proving P(k+1)P(k+1)P(k+1) We want to prove that (k+1)!≤kk+1(k+1)! \le k^{k+1}(k+1)!≤kk+1. Let's start with the left-hand side of the inequality for P(k+1)P(k+1)P(k+1):

    (k+1)!=(k+1)×k!(k+1)! = (k+1) \times k! (k+1)!=(k+1)×k!

    Using our Inductive Hypothesis (k!≤(k−1)kk! \le (k-1)^kk!≤(k−1)k), we can substitute to get:

    (k+1)!≤(k+1)×(k−1)k(k+1)! \le (k+1) \times (k-1)^k (k+1)!≤(k+1)×(k−1)k

    Now, we need to show that this result is less than or equal to the right-hand side of P(k+1)P(k+1)P(k+1), which is kk+1k^{k+1}kk+1. We must prove:

    (k+1)(k−1)k≤kk+1(k+1)(k-1)^k \le k^{k+1} (k+1)(k−1)k≤kk+1

    Let's divide both sides by kkk^kkk. This is allowed as k≥3k \ge 3k≥3.

    (k+1)(k−1)kkk≤k(k+1)\frac{(k-1)^k}{k^k} \le k (k+1)kk(k−1)k​≤k

    (k+1)(k−1k)k≤k(k+1)\left(\frac{k-1}{k}\right)^k \le k (k+1)(kk−1​)k≤k

    (k+1)(1−1k)k≤k(k+1)\left(1 - \frac{1}{k}\right)^k \le k (k+1)(1−k1​)k≤k

    From Bernoulli's inequality or the binomial expansion, we know that for k≥1k \ge 1k≥1, (1−1k)k<1e\left(1 - \frac{1}{k}\right)^k < \frac{1}{e}(1−k1​)k<e1​. Also, (1−1k)k\left(1 - \frac{1}{k}\right)^k(1−k1​)k is an increasing function. Its value at k=3k=3k=3 is (2/3)3=8/27≈0.296(2/3)^3 = 8/27 \approx 0.296(2/3)3=8/27≈0.296. Another known inequality is that for x>1x > 1x>1, (1−1/x)x<1/e<1/2(1 - 1/x)^x < 1/e < 1/2(1−1/x)x<1/e<1/2. For k≥3k \ge 3k≥3, we have:

    (k+1)(1−1k)k<k+1e(k+1)\left(1 - \frac{1}{k}\right)^k < \frac{k+1}{e} (k+1)(1−k1​)k<ek+1​

    To complete the proof, we would need to show k+1e≤k\frac{k+1}{e} \le kek+1​≤k, which means k+1≤ekk+1 \le ekk+1≤ek, or 1≤(e−1)k1 \le (e-1)k1≤(e−1)k. Since e−1≈1.718e-1 \approx 1.718e−1≈1.718 and k≥3k \ge 3k≥3, this inequality holds. Thus, P(k+1)P(k+1)P(k+1) is true.

  4. Conclusion The base case is true, and the inductive step has been proven. By the Principle of Mathematical Induction, n!≤(n−1)nn! \le (n-1)^nn!≤(n−1)n for all n≥3n \ge 3n≥3.

(b) [Challenge] Can you use a similar proof to show that n!≤(n−2)nn! \le (n-2)^nn!≤(n−2)n?

Assumption on nnn

Let's first test for which values of nnn the inequality n!≤(n−2)nn! \le (n-2)^nn!≤(n−2)n might hold.

  • For n=3n=3n=3: 3!=63! = 63!=6, but (3−2)3=13=1(3-2)^3 = 1^3 = 1(3−2)3=13=1. 6≤16 \le 16≤1 is False.
  • For n=4n=4n=4: 4!=244! = 244!=24, but (4−2)4=24=16(4-2)^4 = 2^4 = 16(4−2)4=24=16. 24≤1624 \le 1624≤16 is False.
  • For n=5n=5n=5: 5!=1205! = 1205!=120, but (5−2)5=35=243(5-2)^5 = 3^5 = 243(5−2)5=35=243. 120≤243120 \le 243120≤243 is True.
  • For n=6n=6n=6: 6!=7206! = 7206!=720, but (6−2)6=46=4096(6-2)^6 = 4^6 = 4096(6−2)6=46=4096. 720≤4096720 \le 4096720≤4096 is True.

It appears we need to make the assumption that n≥5n \ge 5n≥5. A similar inductive proof could be attempted with n=5n=5n=5 as the base case, but the algebraic manipulation in the inductive step would be more challenging.

How far can this be pushed?

This question is about finding a tighter upper bound for n!n!n! in the form f(n)nf(n)^nf(n)n. A very important result in mathematics that gives an approximation for the factorial function is Stirling's Approximation:

n!≈2πn(ne)nn! \approx \sqrt{2\pi n} \left(\frac{n}{e}\right)^n n!≈2πn​(en​)n

From this approximation, we can see that n!n!n! grows roughly on the order of (ne)n(\frac{n}{e})^n(en​)n. This suggests that the best integer-based function f(n)f(n)f(n) would be related to ne\frac{n}{e}en​. Since e≈2.718e \approx 2.718e≈2.718, we can see why (n−2)n(n-2)^n(n−2)n is a reasonable bound (for n≥5n \ge 5n≥5), and (n−3)n(n-3)^n(n−3)n would likely fail, as f(n)=n−3f(n)=n-3f(n)=n−3 would be a much smaller base than n/en/en/e.

Therefore, the best upper bound of the form f(n)nf(n)^nf(n)n would involve a function f(n)f(n)f(n) that is close to ne\frac{n}{e}en​.
